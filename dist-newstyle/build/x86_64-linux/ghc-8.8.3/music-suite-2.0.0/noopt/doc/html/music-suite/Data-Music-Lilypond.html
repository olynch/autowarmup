data Value Source #

Value of a \set command. These are simply wrappers for showable things.

For example use (with OverloadedStrings)

Set "Staff.instrumentName" "Violin I"
Set "Staff.instrumentName" 2

to generate

\set Staff.instrumentName = "Violin I"
\set Staff.instrumentName = 2
