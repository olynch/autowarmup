Description

Scales and chords.

Semantically is little distinction between a Scale and a Chord. Thus the Chord and Scale types are synonyms. We use a newtype wrapper to get a more idiomatic Inspectable instance.

Semantically Chord p and Scale p are countable subsets of some pitch space p.

A Mode (or 'ChordType) is like a Chord/Scale that has forgotten its origin.

generator :: ScaleChord o r v p -> NonEmpty v Source #

Returns the interval sequenc that generates the given mode.

The length of this sequence gives the number of notes in the scale, e.g. for a pentatonic (5-note) scale s, length (generator s) == 5, for a heptatonic (7-note) scale it is 7, and so on.
