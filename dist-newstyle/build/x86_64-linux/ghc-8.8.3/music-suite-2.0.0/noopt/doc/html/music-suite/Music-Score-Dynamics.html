level :: Attenuable level dyn a => dyn -> a -> a Source #

Set dynamic level, overwriting previous value.

louder :: Attenuable level dyn a => level -> a -> a Source #

Increase dynamics (linear). For standard notation, switches dynamic marks the given number of step upwards.

>>> louder 1 (mp :: Dynamics)
_p

>>> louder 1 (ff :: Dynamics)
fff

>>> louder 5 [ppp,mf :: Dynamics]
[_f, fffff]

softer :: Attenuable v d a => v -> a -> a Source #

Decrease dynamic (linear). For standard notation, switches dynamic marks the given number of step downwards.
