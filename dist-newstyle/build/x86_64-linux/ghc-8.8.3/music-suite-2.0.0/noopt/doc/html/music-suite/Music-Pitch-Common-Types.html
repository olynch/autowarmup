data Number Source #

The number component of an interval (fourth, fifth) etc. May be negative, indicating a downward interval.

In keeping with music theory tradition, numbers are 1-indexed: the number 1 (or unison) represents an interval of 0 steps, 2 (or second) reprents an interval of 1 step, and so on.
